## Installation

Clone the project to local system and open the current folder

```bash
cd .\stackoverflow\
npm install
```
Again go the main folder 
```bash
cd .\server\
npm install
```

## Start react and server 


```bash
cd .\stackoverflow\
npm start
```

Again go the main folder 
```bash
cd .\server\
npm start
```
