## Installation

Clone the project to local system and open the current folder

```bash
cd .\client\
npm install
```
Again go the main folder 
```bash
cd .\server\
npm install
```

## Start react and server 


```bash
cd .\client\
npm start
```

Again go the main folder 
```bash
cd .\server\
npm start
```

-Database connection is already done and .env file is also included
